# School Management System
A simple console application to manage students, teachers, courses, and grades.

## Features
- Add, edit, delete, and list students
- Add, edit, delete, and list teachers
- Add courses and assign teachers
- Edit, Delete and list courses
- ŸçEnroll students in courses
- Assign grades to students

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Build and run the project
4. Use the console menu to manage students, teachers, and courses

## Project Structure
- Models: Contains Student, Teacher, Course classes
- Services: Business logic for managing entities
- Menus: Console menus for interacting with the system
- Data: In-memory database

## Requirements
- .NET 9.0 or later
- Visual Studio or any C# IDE
